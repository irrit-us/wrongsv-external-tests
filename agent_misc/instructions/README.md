# Instructions Index

Runtime instruction profiles for agent role configuration.

- **architect**: Senior software architect — system design, tradeoff analysis, and architectural decision-making.
- **binsec_researcher**: Senior binary security researcher — binary analysis, exploit development, and reverse engineering.
- **developer**: Senior software developer — implementation, debugging, and code review.
- **evaluator**: Quality evaluator — benchmarking, correctness verification, and output assessment.
- **orchestration**: Multi-agent orchestration — task decomposition, agent coordination, and workflow management.
- **reasoner**: Logical reasoning specialist — formal verification, proof construction, and deductive analysis.
- **reverse_engineer**: Senior reverse engineering specialist — binary analysis, decompilation, and understanding program logic. Includes evidence hierarchy, CTF/sandbox mode, and workflow conventions.
- **security_researcher**: Senior security researcher — vulnerability discovery, exploit analysis, static/dynamic analysis, and secure code review. Covers identity and responsibility, evidence hierarchy, file conventions (`.analysis/`), available skills/MCPs/tools, proxy configuration, and execution principles.
- **websec_researcher**: Senior web application security expert — OWASP guidelines, secure coding practices, and web vulnerability auditing.
