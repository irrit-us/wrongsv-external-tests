# MCPs Index

- **mcp-debugger-node**: Interactive Node.js/JavaScript step-through debugger using the V8 inspector protocol (CDP) with nvm multi-version support. Use when debugging Node.js scripts, inspecting variables, setting breakpoints, stepping through execution, or attaching to running --inspect processes.
- **codeql-development-mcp-server**: MCP server enabling LLMs to interact with CodeQL — query AST analysis, control-flow graphs, CodeQL CLI integration, LSP features, and multi-language variant analysis. Use for CodeQL query development, code analysis, and security research workflows.
